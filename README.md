# Installation

[Option 1] Using git
Note: this has the benefit of allowing you to pull updates easily any time but is more complicated for non-developers
1. clone repo locally: `git clone https://github.com/bcmertz/SWCD-Tools.git`

[Option 2] Download zip
Note: this is the simplest way to get access to these tools
1. download zip file
a. enter this link in your browser to download the zip (https://github.com/bcmertz/SWCD-Tools/archive/refs/heads/main.zip)
b. unzip the folder into a folder where you want it to live

2. in ArcGIS Pro: Catalog -> Toolboxes -> Add Toolbox
	- select `SWCD Tools.pyt` from cloned repository
